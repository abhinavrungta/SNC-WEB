The command line version of the implementation can be run easily using the run.sh script available in the src/ folder.
It asks the user to give input of 4 parameters.
1. Absolute Path of your Dataset. It should be comma seperated with 4 values userId, movieId, rating, timestamp.
2. model to be used when running.
3. Seed Set Size
4. No. of MonteCarlo Simulations to be run.
